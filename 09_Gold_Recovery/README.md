##Подготовка прототипа модели для металлообрабатывающего предприятия
Подготовка прототипа модели машинного обучения, предсказывающей коэффициент восстановления золота из золотосодержащей руды для оптимизации производства
## Используемые инструменты
Python, Pandas, Numpy, Math, Matplotlib, Seaborn, Sklearn, CatBoost, предобработка данных, исследовательский анализ данных, нормализация данных, обучение моделей машинного обучения с подбором гиперпараметров, проверка качества моделей, проверка на адекватность
